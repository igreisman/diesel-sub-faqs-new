# Diesel-Electric Submarine FAQs

Welcome to the MkDocs-powered knowledge base for the Diesel-Electric Submarine FAQ project. Use the navigation to browse the WWII submarine FAQ content, deployment runbooks, and markdown tooling resources.

## What you will find
- Detailed FAQ chapters on hulls, operations, crew life, and combat actions.
- Deployment and hosting guides for PlanetScale, HostGator, and local development.
- Markdown integration notes plus editor demos and sample FAQ content.

## Working locally
1. Install MkDocs with `pip install mkdocs` (or `pipx install mkdocs`).
2. From the repository root, start a live preview: `mkdocs serve`.
3. Build the static site when ready to publish: `mkdocs build`.
